# Image Assets

GitHub 웹에서 이 폴더에 아래 파일명을 그대로 사용해 이미지를 업로드합니다.

| File | Use |
|---|---|
| `ise-logo.png` | 상세 페이지 상단 배너와 메인 Experience 카드 대표 이미지 |
| `presentation-stage.jpg` | 발표 전경 및 발표 내용 섹션 대표 사진 |
| `presentation-closeup.jpg` | 발표자 근접 사진 |
| `presentation-title-slide.jpg` | 발표 제목 슬라이드가 보이는 사진 |
| `conference-badge.jpg` | 프로그램북과 참가 명찰 사진 |

파일이 업로드되기 전에는 페이지에 깨진 이미지 아이콘 대신 안내용 placeholder가 표시됩니다. 같은 파일명으로 업로드하면 별도의 HTML 수정 없이 자동으로 실제 이미지가 나타납니다.
